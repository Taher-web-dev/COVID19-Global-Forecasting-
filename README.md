# Kaggle project  : COVID19-Global-Forecasting-
Kaggle competition intend to forecast the daily spread of COVID-19 in many regions accross the world.
This challenge intend to  forecast confirmed cases and fatalities between March 25 and April 22 by region. 
Thus we had made an explorations and statistical analyses by regions over course the anterior period.
We provided finaly a model which would predict confirmed cases and fatalities for the period between March 25 and April 22.

### Install

To get a local copy up and running follow these simple example steps.
- Open terminal
- Clone this project by the command: 

```
$ git clone git@github.com:Taher-web-dev/COVID19-Global-Forecasting-.git
```

- Then go to the main folder using the next command:

```
$ cd COVID19-Global-Forecasting-
```
